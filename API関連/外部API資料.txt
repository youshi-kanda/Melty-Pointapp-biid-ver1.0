#### 管理のヒント

* **チケット化**：Asana / Jira で上記 18 件をエピック「外部 API / キー取得」にまとめ、担当と期日を設定。
* **週次レビュー**：毎週月曜 AM にステータス確認。期日遅延は即リマインド。
* **成果物保管**：取得したキー／証明書は AWS Secrets Manager、資料は Confluence 等にアップロードし権限管理。

このスケジュールを守れば、**7 月 25 日 Web 版リリース**に必要な外部連携の前提がそろいます。
